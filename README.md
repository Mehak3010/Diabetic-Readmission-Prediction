# Clinical Readmission Risk Prediction using Electronic Health Records (EHR)

## Objective
Predict 30-day hospital readmission for diabetic patients using machine learning.

## Project Structure
- data/ : dataset (place diabetic_data.csv here)
- images/ : exported plots
- model/ : trained model(s)
- diabetic_readmission_prediction.ipynb : main notebook
- requirements.txt : Python dependencies

## Workflow
1. Business Understanding
2. Data Loading
3. Data Quality Assessment
4. Exploratory Data Analysis
5. Feature Engineering
6. Preprocessing
7. Model Training
8. Model Evaluation
9. Explainability (SHAP)
10. Conclusions
